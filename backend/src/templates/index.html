Introduction

Welcome to the API documentation. This API allows you to interact with our service programmatically. Below you will find details on how to authenticate and use the various endpoints available.

Authentication

To use the API, you need to authenticate using an API key. Include the API key in the header of your requests.

Authorization: Bearer YOUR_API_KEY

Endpoints

GET /users

Retrieve a list of users.

GET /users

Response

{
    "users": [
        {
            "id": 1,
            "name": "John Doe",
            "email": "john.doe@example.com"
        },
        {
            "id": 2,
            "name": "Jane Doe",
            "email": "jane.doe@example.com"
        }
    ]
}

POST /users

Create a new user.

POST /users

Request Body

{
    "name": "New User",
    "email": "new.user@example.com"
}

Response

{
    "id": 3,
    "name": "New User",
    "email": "new.user@example.com"
}
